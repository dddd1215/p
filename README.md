# 포토샵기능 메모

```
ctrl + t --> 변형툴
alt + 마우스 휠 조정 --> 화면크기 조절
ctrl + 0 --> 화면에 맞추기
ctrl + 1 --> 화면 100% 채우기
ctrl + j --> 레이어복사
alt + 레이어 드래그 --> 레이어 복사
```
